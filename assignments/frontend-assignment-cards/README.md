# 📘 Assignment: Frontend Foundations - Student Assignment Cards

## 🎯 Objective

Build a simple webpage that displays school assignments as clean visual cards using semantic HTML and basic CSS. You will practice page structure, layout, spacing, and readable design.

## 📝 Tasks

### 🛠️ Build the Page Structure

#### Description
Use the provided starter HTML file to create a page with a clear heading and a section that contains assignment cards.

#### Requirements
Completed program should:

- Add a page title and a main heading for the assignment portal.
- Create at least 3 assignment cards in the HTML.
- Each card must include: assignment title, short description, and due date text.


### 🛠️ Style the Assignment Cards

#### Description
Apply CSS to make the page visually organized and easy to read.

#### Requirements
Completed program should:

- Style each card with padding, border radius, and a subtle shadow.
- Use spacing and font sizes to create clear visual hierarchy.
- Arrange cards in a responsive layout that works on desktop and mobile.
- Add a hover effect to cards (for example, slight lift or color change).
